解决无root权限

显示无root权限，如想获得root权限可以在控制中心选择进入开发者模式


设置–>通用–>开发者选项–>微信登录–>绑定手机号–>输入锁屏密码–>不重启

重新运行 sudo su

使用 UOS 官方软件源

先更新系统：

查看是否有 python3.11 包：

你在 UOS 里 apt search python3.11，搜索结果里有：

  • python311.runtime.szbt → Python3.11 的运行环境
  • python311.devel.szbt → Python3.11 的开发头文件和工具
  • 这两个包安装位置是 /opt/deepin-shared-libs/python311

说明 统信 UOS 已经提供了 Python 3.11（但是不在系统默认路径下）。
