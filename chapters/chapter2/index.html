一、认识print

1. 官方文档：

Python官方文档： https://docs.python.org/zh-cn/3/library/functions.html#print

2. 功能：

从文档可以看到，print 是Python内置的函数， 它可以把它接收到的对象打印出来。

3. stdout是什么？

stdout（标准输出）是程序向外界输出信息的默认通道。当我们使用print函数时， 输出的内容会显示在终端或命令行界面上。

提示： print函数是Python中最常用的函数之一，用于调试和显示程序结果。

二、print的应用

1. 基本用法：

最简单的print用法就是直接输出文本或数值：

# 输出字符串
print("Hello, World!")
print("Python教程")

# 输出数字
print(123)
print(3.14)

# 输出变量
name = "张三"
print(name)
Hello, World! Python教程 123 3.14 张三

2. 打印多个值：

print函数可以一次性输出多个值，用逗号分隔：

# 打印多个值
print("姓名:", "张三", "年龄:", 25)
print("Python", "很", "有趣")

# 使用变量
name = "李四"
age = 20
score = 95.5
print("学生信息:", name, age, score)
姓名: 张三 年龄: 25 Python 很 有趣 学生信息: 李四 20 95.5

3. 格式化输出：

Python提供了多种字符串格式化的方法：

name = "王五"
age = 18
score = 88.5

# 方法1：f字符串（推荐，Python 3.6+）
print(f"我的名字是{name}，今年{age}岁，考试得了{score}分")

# 方法2：format方法
print("我的名字是{}，今年{}岁，考试得了{}分".format(name, age, score))

# 方法3：%格式化（较老的方法）
print("我的名字是%s，今年%d岁，考试得了%.1f分" % (name, age, score))
我的名字是王五，今年18岁，考试得了88.5分 我的名字是王五，今年18岁，考试得了88.5分 我的名字是王五，今年18岁，考试得了88.5分
注意： f字符串（f-string）是Python 3.6+的新特性，是最推荐的字符串格式化方式， 语法简洁且性能优秀。

三、print参数详解

1. sep参数（分隔符）：

sep参数用于指定多个值之间的分隔符，默认是空格：

# 默认分隔符是空格
print("苹果", "香蕉", "橙子")

# 使用逗号作为分隔符
print("苹果", "香蕉", "橙子", sep=", ")

# 使用连字符作为分隔符
print("2024", "12", "25", sep="-")

# 使用空字符串作为分隔符
print("Hello", "World", sep="")
苹果 香蕉 橙子 苹果, 香蕉, 橙子 2024-12-25 HelloWorld

2. end参数（结尾字符）：

end参数用于指定print输出后的结尾字符，默认是换行符：

# 默认end是换行符\n
print("第一行")
print("第二行")

print("---分隔线---")

# 使用空格作为结尾
print("Hello", end=" ")
print("World")

print("---分隔线---")

# 使用其他字符作为结尾
print("加载中", end="")
print(".", end="")
print(".", end="")
print(".", end="")
print(" 完成!")
第一行 第二行 ---分隔线--- Hello World ---分隔线--- 加载中.... 完成!

3. file参数（输出目标）：

file参数用于指定输出的目标，默认是标准输出：

import sys

# 输出到标准输出（默认）
print("输出到控制台")

# 输出到标准错误
print("这是错误信息", file=sys.stderr)

# 输出到文件
with open("output.txt", "w", encoding="utf-8") as f:
    print("这行文字会写入文件", file=f)
提示： file参数在日志记录和错误处理中很有用。

四、实际应用示例

1. 制作简单的表格：

# 打印表格头
print("姓名", "年龄", "成绩", sep="\t")
print("-" * 20)

# 打印表格数据
print("张三", 18, 95, sep="\t")
print("李四", 19, 87, sep="\t")
print("王五", 17, 92, sep="\t")
姓名 年龄 成绩 -------------------- 张三 18 95 李四 19 87 王五 17 92

2. 制作进度条效果：

import time

# 模拟进度条
for i in range(11):
    print(f"\r进度: [{'=' * i}{' ' * (10-i)}] {i*10}%", end="")
    time.sleep(0.5)
print("\n任务完成！")

3. 调试信息输出：

def debug_info(variable_name, value):
    """输出调试信息"""
    print(f"[DEBUG] {variable_name} = {value} (类型: {type(value).__name__})")

# 使用示例
x = 10
y = "Hello"
z = [1, 2, 3]

debug_info("x", x)
debug_info("y", y)
debug_info("z", z)
[DEBUG] x = 10 (类型: int) [DEBUG] y = Hello (类型: str) [DEBUG] z = [1, 2, 3] (类型: list)

五、常见问题和注意事项

注意：
  • print函数在Python 2.x中是语句，在Python 3.x中是函数
  • 输出中文时注意编码问题，建议使用UTF-8编码
  • 大量输出可能影响程序性能，生产环境中要谨慎使用
